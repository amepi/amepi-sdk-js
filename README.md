# sdk-js

JavaScript SDK for Amepi softwares.

## Features

- It works in node.js (AWS Lambda), deno and web browsers (Google Chrome)
- Zero dependencies
- Drizzle ORM:
  - Database schema and migrations
  - Zod schema generation and validation
